### Get Started a Project

1. **Install dependencies**

```sh
  npm i
```

2.  **Run in local DEV mode at http://localhost:5173/ host:**

```sh
  npm run dev
```

3. **Build the project to /dist directory:**

```sh
  npm run build
```

4. **Run in local preview mode at http://localhost:4173/ host:**

```sh
  npm run preview
```
